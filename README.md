# Miercoles

**Pronunciation:** /mi-ER-ko-les/ (Spanish for "Wednesday")

A whimsical pet name generator inspired by Gomez Addams' impossibly adorable nicknames for his daughter Wednesday.

## Inspiration

This project was born from binge-watching Netflix's _Wednesday_ and falling in love with Gomez Addams' over-the-top terms of endearment for his daughter. The way he greets her with names like "my little tormenta" and "my little death trap" perfectly captures my newfound feline fatherhood.

Turns out, "Hi my little nightmare" hits different when you're addressing an 8-pound Siamese mix who play-fights with your feet at 7am to request breakfast. ðŸ˜¸

## Features

- **Endless Pet Names** - Tap anywhere to generate a new ridiculous nickname from a curated collection
- **Beautiful Typography** - Custom font with elegant radial gradient text effects
- **Interactive Hover Effects** - Gradient follows your cursor on desktop and touch on mobile
- **Copy to Clipboard** - Tap the nickname to copy the full phrase "Hi my little [nickname]"
- **Music Player** - Ambient background music with play/pause controls
- **Circular Audio Visualizer** - Radial frequency visualization that pulses with the music
- **Fullscreen Mode** - Press and hold the music button to expand into an immersive visualizer experience
- **Session Persistence** - Music preference is remembered between visits
- **Responsive Design** - Works beautifully on all screen sizes

## Development

```bash
# Install dependencies
bun install

# Start development server
bun run dev

# Build for production
bun run build

# Preview production build
bun run preview
```
